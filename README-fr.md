## 📚 Résumé

African Universities Registry (AUR) est une plateforme publique et fiable qui recense et met à jour les établissements d'enseignement supérieur reconnus en Afrique. Le projet débute avec le Togo et vise à s'étendre progressivement à tout le continent africain.

Cette initiative facilite l'orientation des étudiants, renforce la transparence dans l'enseignement supérieur, et fournit des outils aux institutions, employeurs et autorités pour la vérification et l'analyse des diplômes.

## 🎯 Problème résolu

Les candidats aux études supérieures et les employeurs peinent à distinguer les établissements accrédités des offres non conformes. Les ministères disposent d'informations officielles mais elles sont souvent dispersées, statiques ou difficilement exploitables par le public.

## 💡 Solution

### Fonctionnalités principales

- **📋 Registre en ligne** : Base de données des établissements reconnus avec profils détaillés (statut, filières, accréditations, contacts)
- **🔌 API ouverte** : Jeux de données (CSV/JSON) pour intégration par plateformes d'emploi, universités et médias
- **✅ Vérification des diplômes** : Système de vérification via QR code ou numéro unique
- **🗺️ Carte interactive** : Visualisation géographique et comparateur de filières (coûts, durée, débouchés)
- **🔔 Système d'alertes** : Notifications (email/SMS/WhatsApp) pour les changements de statut
- **📊 Tableau de bord** : Interface pour autorités avec suivi qualité et indicateurs

### Technologies prévues

- **Backend** : Node.js/Express ou Python/Django
- **Base de données** : PostgreSQL avec versioning par année académique
- **Frontend** : React.js ou Vue.js
- **API** : RESTful API avec documentation OpenAPI
- **Mobile** : Application mobile responsive
- **Géolocalisation** : Integration avec services de cartographie
- **Sécurité** : Authentification, chiffrement des données sensibles

## 🎯 Bénéfices clés

### Pour les étudiants et parents

- Orientation fiable et protection contre les écoles non agréées
- Comparaison transparente des programmes et coûts
- Accès à l'information officielle centralisée

### Pour les employeurs

- Vérification rapide et standardisée des diplômes
- Réduction des risques de fraude académique
- Base de données fiable pour le recrutement

### Pour les ministères

- Pilotage et contrôle qualité facilités
- Publication automatisée des données officielles
- Amélioration de la transparence

### Pour les établissements

- Visibilité accrue auprès du public
- Concurrence saine et amélioration continue
- Validation officielle de leur statut
