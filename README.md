## 📚 Overview

African Universities Registry (AUR) is a reliable public platform that catalogs and maintains up-to-date information about recognized higher education institutions in Africa. The project starts with Togo and aims to progressively expand across the African continent.

This initiative facilitates student guidance, strengthens transparency in higher education, and provides tools for institutions, employers, and authorities for diploma verification and analysis.

## 🎯 Problem statement

Students applying to higher education and employers struggle to distinguish accredited institutions from non-compliant offerings. Ministries have official information but it is often scattered, static, or difficult for the public to access and use.

## 💡 Solution

### Core features

- **📋 Online Registry**: Database of institutions with detailed profiles (programs, contacts, public information)
- **📊 JSON Datasets**: Open data in JSON format for integration by platforms and developers
- **🗺️ Interactive Map**: Geographic visualization and program comparator (costs, duration, career prospects)
- **📈 Public Dashboard**: Analytics and statistics about higher education landscape
- **🔍 Search & Filter**: Advanced search by location, programs, costs, and other criteria

### Planned technology stack

- **Backend**: Hono TypeScript
- **Database**: PostgreSQL
- **Frontend**: Next.js (with i18n support for EN/FR UI)
- **Data Format**: JSON datasets
- **Mobile**: Responsive web application
- **Geolocation**: Integration with mapping services
